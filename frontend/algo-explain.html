投资组合健康评分

核心算法

算法目的

综合评估用户投资组合的多元化程度、风险敞口、现金配置等因素，生成 0-100 分的健康评分。

计算公式

综合得分 = Σ(维度得分 × 维度权重)
// 各维度满分100，权重总和为1

评分维度

  • 多元化得分 基于行业分散度计算，单一行业占比超过40%开始扣分
  • 风险敞口 评估个股集中度，单只股票占比超过25%视为高风险
  • 现金配置 对比用户设定的目标现金比例，偏离度越大扣分越多
  • 盈利能力 持仓整体盈亏状态，浮亏超过10%开始影响得分
计算示例

多元化 85分(权重25%) + 风险敞口 70分(权重25%) + 现金配置 90分(权重25%) + 盈利能力 80分(权重25%)

综合得分 = 85×0.25 + 70×0.25 + 90×0.25 + 80×0.25 = 81.25分

期权 Sell Put 策略推荐

期权算法

算法逻辑

基于参考投资体系中的 Sell Put 核心原则：只在具有强护城河、长期看好的标的上卖出 Put，结合实时股价、VIX 水平和用户风险偏好，动态生成推荐。

推荐标的（固定5个，优先级排序）

  • TSLA 电动车/能源/AI机器人 — 核心持仓，Sell Put 首选；隐含波动率估算 55%，OTM 倍数 1.0×
  • NVDA AI 算力基础设施 — 护城河极宽；隐含波动率估算 50%，OTM 倍数 1.1×（稍宽）
  • GOOGL AI/搜索/云计算 — AI 巨头中估值最低；隐含波动率估算 35%，OTM 倍数 0.9×
  • META AI 社交/广告双引擎 — 被低估；隐含波动率估算 38%，OTM 倍数 0.9×
  • IONQ 量子计算（高弹性）— 高风险高回报，建议小仓位；隐含波动率估算 80%，OTM 倍数 1.4×（更远）

OTM 距离计算

基础 OTM% = 保守15% / 平衡10% / 激进5%（由用户画像决定）
实际 OTM% = 基础 OTM% × 标的 OTM 倍数
行权价 = 现价 × (1 − 实际 OTM%)
// 行权价向下取整到 $5/$2.5/$0.1 精度（依价格区间）

权利金估算（简化 Black-Scholes）

权利金 ≈ 现价 × IV × √(DTE/365) × Delta目标
// IV = 各标的隐含波动率估算（TSLA 0.55，IONQ 0.80 等）
// Delta目标：保守 0.15，平衡 0.25，激进 0.35
// 上限为现价的 10%，下限 $0.10

年化收益率计算

年化收益率 = (权利金 / 行权价) × (365 / DTE) × 100%

DTE（到期天数）与胜率

  • 保守型 DTE = 45天，胜率 85%，OTM 基础 15%
  • 平衡型 DTE = 30天，胜率 75%，OTM 基础 10%
  • 激进型 DTE = 21天，胜率 65%，OTM 基础 5%

VIX 调节说明

  • VIX < 15 波动率低，权利金偏少，建议等待更好时机
  • VIX 15–30 正常区间，按标准参数操作
  • VIX > 30 权利金丰厚，此时 Sell Put 收益更高

价格数据来源

推荐标的的现价通过多源实时获取：优先使用 Stooq（免费、无限速），失败时降级至 Alpha Vantage。两源差异 > 10% 时触发价格警告。

计算示例（用户画像：平衡型）

TSLA 实时价 $361.75，OTM = 10% × 1.0 = 10%

行权价 = 361.75 × 0.90 ≈ $325（取整到 $5）

权利金 ≈ 361.75 × 0.55 × √(30/365) × 0.25 ≈ $14.5/股

年化收益 = (14.5 / 325) × (365/30) × 100% ≈ 54.2%

注：实际成交价受市场供需影响，此为算法估算值，仅供参考

风险预警系统

风控算法

预警触发条件

  • 现金比例过高 实际现金比例超过目标比例 +15%
  • 行业集中度 单一行业持仓占比超过 50%
  • 个股集中度 单只股票占比超过 30%
  • 浮亏预警 单只股票浮亏超过 -15%
  • VIX 飙升 VIX 单日涨幅超过 20% 或突破 30

预警等级

高风险（红色）：需要立即关注，可能影响整体组合安全

中风险（黄色）：建议关注，适时调整

提示（蓝色）：信息性提醒，供参考

持仓盈亏计算

基础计算

正股盈亏

浮动盈亏 = (当前价 - 成本价) × 持股数量
盈亏比例 = (当前价 - 成本价) / 成本价 × 100%

期权盈亏

持仓中（未平仓）：

最大可收益 = 权利金 × 合约数 × 100股

到期归零：

已实现盈亏 = 权利金 × 合约数 × 100股（全部收入）

提前平仓：

Sell Put 盈亏 = 收取权利金 - 平仓成本
// 平仓成本 = 买回价格 × 合约数 × 100股
