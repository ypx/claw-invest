Sell Put 期权策略
专属策略体系

专为保守型投资者设计，利用高现金比例执行 Sell Put 策略，以低风险方式增加现金流。

Sell Put 核心原则
适用条件
VIX 20-35 区间 · 高概率 OTM · 目标获利 25-50% 了结
仓位原则
单笔不超过账户 5% · 保持充足现金作保证金
风险管理
亏损超过 2x 权利金止损 · 避开财报前两周
想定价买入股票？
用 Sell Put（卖出看跌期权）

💰 先收取权利金
📉 如果股价跌到行权价以下，你以低价买入股票
📈 如果股价上涨，权利金白赚
最常用 适合抄底
想高价卖出持股？
用 Sell Call（卖出看涨期权）= Covered Call

💰 持有股票的同时卖出 CALL，收取权利金
📈 如果股价大涨，你以高价卖出（行权价）
📉 如果股价不涨，权利金白赚
增强收益 适合震荡市
想保护持仓防下跌？
用 Protective Put（保护性看跌期权）

🛡️ 买入看跌期权给股票上保险
📉 股价下跌时，看跌期权盈利抵消损失
📈 股价上涨时，依然享受全部涨幅
防守型 保险策略
简单选择原则
✅ 想低价买入 → Sell Put（当前主要策略）
✅ 有正股想额外赚钱 → Sell Call（Covered Call）
✅ 担心持仓下跌 → Protective Put
⏱️ 期权周期：建议 30-45 天到期，时间价值衰减最快
