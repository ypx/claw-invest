## 2026-03-30 语言切换按钮失效紧急修复

### 问题描述
语言切换按钮点击无反应：
- 登录/注册页面：点击语言切换按钮（EN/中文）无任何反应
- Dashboard主页面：点击语言切换按钮（中/EN）无任何反应

### 根本原因
按钮点击处理程序直接调用 `i18n.switchLanguage()`，没有安全检查。当i18n对象未完全初始化或存在JavaScript错误时，会导致按钮失效。

### 修复详情

**1. Dashboard页面（claw_dashboard_enhanced.html）**

添加安全包装函数：
```javascript
function safeSwitchLanguage(lang) {
  if (!window.i18n) {
    console.error('i18n not loaded');
    alert('语言切换功能初始化中，请刷新页面重试');
    return;
  }
  if (!window.i18n.switchLanguage) {
    console.error('i18n.switchLanguage not available');
    alert('语言切换功能未准备好，请刷新页面重试');
    return;
  }
  window.i18n.switchLanguage(lang).catch(err => {
    console.error('Language switch failed:', err);
    alert('语言切换失败：' + err.message);
  });
}
```

修改按钮onclick：
```html
<button onclick="safeSwitchLanguage('zh')" class="lang-btn active">中</button>
<button onclick="safeSwitchLanguage('en')" class="lang-btn">EN</button>
```

**2. 登录页面（claw_login.html）**

添加安全包装函数：
```javascript
function safeLoginSwitchLanguage() {
  if (!window.i18n) {
    console.error('i18n not loaded');
    alert('语言切换功能初始化中，请刷新页面重试');
    return;
  }
  if (!window.i18n.getCurrentLang || !window.i18n.switchLanguage) {
    console.error('i18n methods not available');
    alert('语言切换功能未准备好，请刷新页面重试');
    return;
  }
  const currentLang = window.i18n.getCurrentLang();
  window.i18n.switchLanguage(currentLang === 'zh' ? 'en' : 'zh').catch(err => {
    console.error('Language switch failed:', err);
    alert('语言切换失败：' + err.message);
  });
}
```

修改按钮点击处理：
```javascript
document.getElementById('lang-switch-btn').addEventListener('click', safeLoginSwitchLanguage);
```
