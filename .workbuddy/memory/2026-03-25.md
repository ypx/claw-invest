## 移动端深度适配修复 v1.3.0（10:35）

### 问题来源
用户反馈：手机端所有页面都向右溢出，需要左滑才能看全

### 根本原因
`#main { margin-left: 240px; }` 在手机端没有重置，导致内容区域向右偏移了240px

### 修复内容

1. **修复#main容器**
   - 添加 `#main { margin-left: 0; width: 100%; max-width: 100vw; overflow-x: hidden; }`
   - 添加 `body { overflow-x: hidden; }` 防止整体溢出

2. **优化KPI卡片布局（数据少时并排）**
   - 投资总览、持仓明细、TSLA、市场行情等页面的KPI统一为2x2网格
   - 480px以下仍保持2x2，只是缩小间距和padding
   - 图标尺寸统一缩小为36x36px

3. **修复内联grid样式**
   - 覆盖所有 `[style*="grid-template-columns:1fr 1fr"]` 为单列
   - 覆盖所有 `[style*="auto-fill"]` 为单列

## 三处UI Bug修复（09:09）

### 问题与修复

1. **期权持仓在投资总览/持仓明细不显示**
   - 根因：`navigate()` 函数切换到 overview/holdings 时未调用 `loadOptionsOverview()`；初次加载若失败后无重试
   - 修复：`navigate()` 里对 `overview` 和 `holdings` 页加 `setTimeout(loadOptionsOverview, 80)`
   - 修复：`loadOptionsOverview` 加 401 跳转登录处理，改善错误日志

2. **健康评估环形图全黑**
   - 根因：`buildHealthRing()` 里用了 CSS 变量（`var(--success)` 等），Chart.js 无法解析 CSS 变量
   - 修复：改为真实颜色值 —— 好（≥80）:`#2e8b57`，中（≥60）:`#b07d1a`，差:`#d94f2a`；背景色 `#ede8e3`，边框色 `#f7f3ef`

3. **风险提示「险」字换行溢出**
   - 根因：`.alert-badge` 固定 `width:22px; height:22px` 圆形，装不下"高风险"三个字
   - 修复：改为自适应宽度胶囊（`height:20px; padding:0 7px; white-space:nowrap; vertical-align:middle`），文字完整显示

## UI 全面修复（08:30）

### 问题来源
用户反馈：BETA 标签重复、Tab 标签 emoji 不协调、Tab 内容不显示（功能失效）

### 已修复

1. **BETA 标签重复** → 删除 sidebar 中多余的 `<div class="sidebar-section"><span class="logo-badge">BETA</span></div>`，只保留 logo 处一个

2. **Tab 按钮 emoji → FA 图标**
   - TSLA 页：🏰护城河→`fa-shield-halved`，🔮业务前景→`fa-binoculars`，⚡Sell Put→`fa-bolt`，🧮收益计算器→`fa-calculator`
   - 持仓管理页：📈正股持仓→`fa-chart-line`，📊期权持仓→`fa-sliders`，⚙️设置→`fa-gear`

3. **Tab CSS 统一重构**
   - 新增 `.tsla-tab-bar` 和 `.manage-tab-bar` container class
   - `.tsla-tab` 和 `.manage-tab` 样式统一，新增 `display:flex; align-items:center; gap:6px;` + icon opacity 控制
   - 激活状态只用 `.active` class 控制，移除 JS 中 inline style 覆盖

4. **Tab 内容不显示 bug（核心 bug）**
   - 原因：CSS `.tsla-tab-content { display: none }` 默认隐藏所有内容，但 switchTslaTab 用 `element.style.display = ''` 清除 inline style，恢复到 CSS 的 `display:none`（而不是显示）
   - 修复：`switchTslaTab` 和 `switchManageTab` 改用 `element.style.display = 'block'`
   - 修复：第一个 tab content（护城河/正股持仓）加 `style="display:block"` 初始可见

5. **JS 中 inline style 覆盖 CSS 激活状态** → switchTslaTab 和 switchManageTab 去掉 `t.style.borderBottomColor` 和 `t.style.color` 操作，只用 classList.add/remove('active')

6. **全面 UI 排版优化**
   - `.page` padding 从 24px 改为 28px 28px 40px（底部更宽松）
   - `.kpi-value` 字号 28px→26px，line-height:1.2（更紧凑）
   - `.kpi-label` 字号 13px→12px
   - `.kpi-change` 字号 13px→12px，新增 `.neutral` 状态（gray）
   - `.section-title` 新增 page 级别大号（16px），普通卡片内 14px
   - `th` padding 优化（12px→10px 12px），白空间合理化
