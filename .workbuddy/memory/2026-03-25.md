## 2026-03-26 实时价格引擎改造

### 核心变更
- 彻底废弃 Alpha Vantage（免费版不支持 INDEX endpoint，且频率限制严格）
- 改用 Yahoo Finance（query1.finance.yahoo.com/v8/finance/chart）
  - 无需 API Key
  - 无频率限制（个人使用）
  - 支持盘前盘后数据
  - SSL 需绕过（macOS 证书链问题）

### 后端改动（backend.py）
- 新增 `_yf_fetch(symbol)` 函数：获取单只股票报价
- 新增 `get_price(symbol)` / `get_prices_batch(symbols)` 带缓存并发批量查询
- 新增 `_fetch_vix_history(days=7)` 获取 VIX 7天历史
- `get_market_data()` 改为真实数据：SPY→S&P500，QQQ→Nasdaq，^VIX→VIX
- `dashboard()` 接口：每次调用时并发获取所有持仓实时价格并写回数据库
- `_calc_risk_alerts()` 改为动态计算（基于真实持仓+VIX+现金比例）
- 新增 `GET /api/quotes/{symbol}` 单只报价接口
- 新增 `GET /api/quotes?symbols=TSLA,NVDA` 批量报价接口

### 前端改动（claw_dashboard_enhanced.html）
- 持仓表格新增「今日涨跌」列（带颜色）
- 市场行情页：数据标注来源+开盘状态（盘中/盘前/盘后/休市）
- VIX 图表改为动态渲染（使用真实7天历史数据）
